### 栅格系统

> 更快捷的实现响应式
>
> **预定义类**    css
>
> 把屏幕平均分成12列
>
> 三种终端  四种屏幕大小   移动端优先



|           | 手机 < 768px | 平板 768~992 | pc小屏 992~1200 | pc大屏 > 1200 |
| :-------: | :--------: | :--------: | :-----------: | :---------: |
| container |    none    |   750px    |     970px     |   1170px    |
|    前缀     |  col-xs-   |  col-sm-   |    col-md-    |   col-lg-   |
|    隐藏     | hidden-xs  | hidden-sm  |   hidden-md   |  hidden-md  |
|           |            |            |               |             |
